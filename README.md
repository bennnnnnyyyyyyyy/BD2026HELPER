# Power Dialer & Hub

A modern, standalone CSV dialer and DME Data Hub.

## Features

-   **CSV Dialer**: Load leads, cycle through them, and track dispositions.
-   **Smart Mapping**: Automatically maps headers like `AuthOfficialName`, `Legalbusinessname`, `AuthPhone`.
-   **Disposition Tracking**: Buttons for VM, GK, NA, WN, and Custom notes.
-   **Comment History**: View past comments and append new notes with timestamps.
-   **Export**: Download your updated CSV with full call history and notes.

## How to Use

1.  Open `homepage.html` to access the hub.
2.  Click "Launch Dialer" to open `dialer.html`.
3.  Drag & Drop your CSV file.
4.  Confirm the column mapping.
5.  Start calling! Use the buttons to log the outcome of each call.
6.  Click "Export CSV" when finished to save your work.

## Files

-   `homepage.html`: The main dashboard.
-   `dialer.html`: The calling application.
